## Всем привет, меня зовут Дмитрий Солдатов 👋

Инженер по нагрузочному тестированию и разработчик инструментов для тестирования сетевой инфраструктуры. Специализируюсь на тестировании производительности средств защиты информации, разработке профилей трафика и автоматизации процессов. Занимаюсь исследованием сетевых протоколов, высокопроизводительных сетевых технологий и реализацией методик нагрузочного тестирования в соответствии с RFC 9411.
